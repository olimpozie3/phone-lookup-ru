# 📞 Поиск по номеру телефона РФ

Бесплатное веб-приложение для определения оператора и региона российских номеров телефонов.

## ✨ Возможности

- 🔍 Определение оператора связи (МТС, МегаФон, Билайн, Теле2, YOTA и др.)
- 📍 Определение региона и города
- 📊 Тип номера (мобильный / городской / федеральный)
- 📋 История поиска (последние 10 запросов)
- 🌙 Тёмная и светлая тема
- 📱 Адаптивный дизайн (мобильный + десктоп)
- 📥 Копирование результата в буфер обмена

## 🚀 Запуск

Просто откройте файл `index.html` в браузере — сервер не нужен!

Или откройте онлайн через GitHub Pages:
👉 https://olimpozie3.github.io/phone-lookup-ru/

## 📡 Источники данных

- [num.voxlink.ru](https://num.voxlink.ru) — реестр нумерации Минцифры РФ
- Встроенная база DEF-кодов операторов (МТС, МегаФон, Билайн, Теле2, YOTA)
- Справочник кодов городов России (60+ городов)
